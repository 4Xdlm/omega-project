# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| v3.161.x | Yes |
| v3.160.x | Yes |
| < v3.160 | No |

## Reporting a Vulnerability

1. **DO NOT** open a public issue
2. Open issue with tag `[SECURITY]` with minimal details
3. Or contact maintainers directly

## Response Time

- Acknowledgment: 48 hours
- Assessment: 7 days
- Fix (if confirmed): 30 days

## What to Include

- Description of vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)
