# OMEGA — Guide de Démarrage Rapide

## Installation (2 minutes)

1. **Télécharger** `OMEGA_Setup_v1.7.0-INDUSTRIAL_x64.exe`
2. **Double-cliquer** sur l'installeur
3. **Suivre** les instructions (installation utilisateur, pas besoin d'admin)
4. **Lancer** OMEGA depuis le menu Démarrer ou le bureau

## Premier Lancement

1. **Ouvrir** OMEGA UI
2. **Coller** ou **importer** votre texte (roman, chapitre, extrait)
3. **Cliquer** sur "Analyser"
4. **Explorer** les résultats :
   - 🎭 Émotions détectées (joie, tristesse, colère, peur...)
   - 📊 Scores par émotion
   - 🔍 Holographe de cohérence

## Fonctionnalités Principales

| Fonction | Description |
|----------|-------------|
| **Analyse Émotionnelle** | Détecte 8 émotions de base (Plutchik) |
| **Lexique FR Gold** | 118 mots-clés français certifiés |
| **Holographe** | Scanner de cohérence narrative |
| **Export** | Markdown et DOCX |

## Raccourcis Clavier

| Raccourci | Action |
|-----------|--------|
| `Ctrl+V` | Coller du texte |
| `Ctrl+O` | Ouvrir un fichier |
| `Ctrl+S` | Sauvegarder le projet |
| `Ctrl+E` | Exporter les résultats |
